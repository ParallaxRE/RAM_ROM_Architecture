# RAM - Random Access Memory (Rastgele Erişimli Bellek)

---

## 🇹🇷 Türkçe Notlar
RAM, işlemcinin o an üzerinde çalıştığı verileri tutan yüksek hızlı, geçici bir çalışma alanıdır.

### Teknik Detaylar:
* **Zamanlama (Timing):** Bir okuma döngüsünde (Read Cycle), adres verildikten sonra verinin hat üzerinde geçerli olması için $t_{CO}$ gibi gecikmelerin aşılması gerekir.
* **Hiyerarşi:** SRAM (Static RAM) işlemci önbelleği (Cache) olarak kullanılırken, DRAM (Dynamic RAM) ana bellek olarak kullanılır.
* **Bit Kaydırma (Shifting):** Donanım seviyesinde en hızlı işlem türüdür; veriyi 2 ile çarpmak veya bölmek için kullanılır.

---

## 🇺🇸 English Notes
RAM is a high-speed, volatile workspace that holds data currently being processed by the CPU.

### Technical Details:
* **Timing:** In a Read Cycle, after the address is provided, delays like $t_{CO}$ must pass before the data becomes valid on the bus.
* **Hierarchy:** SRAM (Static RAM) is used as CPU Cache, while DRAM (Dynamic RAM) serves as the main system memory.
* **Bit Shifting:** The fastest type of operation at the hardware level; used to multiply or divide data by powers of 2.
