# ROM - Read Only Memory (Sadece Okunabilir Bellek)

### 🇹🇷 Türkçe Açıklama
ROM, verilerin kalıcı olarak saklandığı bellek türüdür. Elektrik kesilse bile veriler silinmez (Non-volatile). 
Bu bölümde ROM mimarisi, fonksiyon jeneratörü olarak kullanımı ve DAC bağlantıları üzerine notlarım yer almaktadır.

### 🇺🇸 English Description
ROM is a type of memory where data is permanently stored. It is non-volatile, meaning data is not lost when power is off. 
This section contains my notes on ROM architecture, its use as a function generator, and DAC connections.
